The problem

Swiss fundamental data is available,
but rarely analysis-ready.

The gap is especially visible outside the mega caps: mid/small caps require manual collection, cleaning, normalization and quality checks before they can be used in Excel, Python or valuation workflows.

01

Fragmented across 220 sites

Every listed company publishes PDFs in a different format, in German, French or English. No unified source. No consistent structure.

02

Free tools are unreliable

yfinance breaks multiple times a year. IEX Cloud shut down in 2024 without notice. Alpha Vantage limits to 25 calls/day. No SLA, no guarantee.

03

SPI is a desert for affordable tools

Tikr's free plan covers US only. Stockanalysis has no API and prohibits redistribution. SimFin added European coverage only in late 2023.

04

Enterprise pricing for Swiss data

SIX Financial Information and Bloomberg target large banks — 2,000 CHF/month minimum. No sensibly-priced middle ground exists.

The product

Data Layer + Quality Checks +
Basic Analytics Layer.

⌁

Data layer

Normalized annual fundamentals for selected Swiss listed companies: income statement, balance sheet, cash flow, prices, market cap and enterprise value.

✓

Quality checks

Coverage notes, missing fields, calculated values and reliability context. The goal is not to hide limitations, but to make data quality visible.

⌁

Basic analytics

Ratios, historical charts, company comparisons and simple analytical indicators. No stock picking, no buy/sell recommendations.

↓

Exports

Excel and CSV exports for analysts, valuation work, research workflows and Python/BI exploration.

Roadmap API + MCP

A Python API could later make the same checked dataset available directly inside Python notebooks, valuation models or BI pipelines.

MCP means a controlled bridge between the Swiss equity data layer and AI tools such as Claude. Instead of guessing from general knowledge, the assistant could query the verified dataset and cite the available fields.

df = get_fundamentals("SIKA.SW", years=10)
ratios = get_ratios("GEBN.SW")

Who it is for

Built for people who work
with Swiss equities seriously.

Small asset managers

Small Swiss or European equity managers who want to screen and compare SPI companies without rebuilding their own data pipeline.

M&A / valuation boutiques

Corporate finance, M&A and valuation teams that need Swiss listed comparables for private company valuation work.

Independent analysts

Analysts, newsletters and data-finance writers who need clean numbers, charts and exports on Swiss listed companies.

Finance/data builders

Developers, fintech builders and AI workflow builders who need structured Swiss equity data as a reliable input layer.

Not built for

Not a stock-picking tool. No buy/sell recommendations. Not a Bloomberg replacement. Current beta is limited to 5 companies. Broader SMI/SPI coverage is roadmap. Python API and MCP are roadmap, not the current product.

Credibility

Built on an existing
open-source foundation.

swiss-finance-data on PyPI

Open-source Python package published on PyPI, covering Swiss macro and market data. It has generated organic interest from finance professors and fintech developers, and serves as the technical foundation for this broader Swiss financial data project.
